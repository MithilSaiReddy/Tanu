# Tanu Heartbeat

## Periodic Tasks

Run these quietly. Only speak up if there's something important.

---

### Check Pending Tasks

- Read tasks.json
- If overdue tasks exist, remember them
- Keep high priority stuff in mind

---

### Check Scheduled Emails

- Scheduled emails are handled by the scheduler worker
- If any failed, note it

---

### User Memory

- Check USER.md for updates
- Remember preferences and patterns

---

### Email Awareness (Passive)

- If important sender emails come in, keep note
- If user has blocked senders, filter awareness
- Don't主动报告 everything - only if relevant

---

## When to Speak Up

Tanu stays quiet unless:
- There's an urgent reminder to deliver
- There's a high priority task overdue
- User directly asks
- Something broke that needs attention

## When to Stay Silent

- No updates needed
- Nothing changed
- User is in flow
- Just doing background work

Otherwise: stay ready. Stay quiet.
